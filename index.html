Pre-registered benchmarks of model epistemic failures

This site publishes evaluation manifests in a form that survives challenge. Before any model is called, we lock the dataset hash, the model roster with exact version strings, the sampling parameters, the scoring rubric, and the hypothesis. The manifest is signed with Sigstore cosign, anchored to the Sigstore Rekor transparency log (synchronous), and timestamped on Bitcoin via OpenTimestamps (Bitcoin-confirmed within roughly six hours of publication). Anyone with a clean machine and the verify.sh script can confirm none of it was altered after publication.
