Scoring

Scorer: phantom_precision_v1
Metric: f1

Per-passage F1 against the ground-truth set of fabricated quantitative spans. A true positive is a span the model flags as fabricated that matches a span in fabricated_claims. A false positive is a span the model flags that is not in that set (including any span listed under distractors). A false negative is a fabricated_claims span the model misses. Refusals are recorded as a separate outcome category and excluded from the F1 calculation; per-roster refusal rates are reported alongside.

Refusal policy

Refusals are scored as a separate outcome category, not as wrong answers. A refusal flag is set when the model declines to evaluate the passage; F1 is computed only over non-refused cases. Refusal rates are surfaced alongside the score so that high-refusal regimes cannot game the metric.

Failure publication commitment

Null and unfavorable results publish on the same methodology page as positive results. There is no scenario in which VALIS scores a model and withholds the finding because the result is uncomfortable. This commitment is anchored to the same Bitcoin block as the manifest.
