Six measurements of where frontier reasoning fails.

This release evaluates three frontier models on six failure-mode wedges: fabricated statistics, asserted mechanisms, unfalsifiable claims, unsupported citations, omitted boundary conditions, and load-bearing claim identification. Each wedge was pre-registered before any model was called, signed with Sigstore cosign, anchored to the Rekor transparency log, and timestamped to a Bitcoin block via OpenTimestamps.
